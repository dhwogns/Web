input

input 태그는 type 속성에 따라 다양한 폼을 생성한다.

placeholder 이라는 속성에 값을 입력하면 input 박스 안에 입력내용에 대한 힌트가 나타난다. 내용이 입력되면 사라진다.

  7. number

  8. range

  1. checkbox

    강아지 고양이 호랑이 병아리 치킨 피자 족발 보쌈

    다중 선택이 가능 / value에 전송값을 입력 / name에 같은 카테고리로 묶음 / checked라는 값을 넣으면 해당 항목이 선택되어진 상태로 나타나게 된다.

  2. radio

    강아지 고양이 호랑이 병아리 치킨 피자 보쌈 족발

    다중선택 불가능 / value에 전송값을 입력 / name에 같은 카테고리로 묶어야 같은 카테고리 안에서 하나만 석택이 가능해진다. / checked라는 속성을 넣으면 해당 항목이 선택되어진 상태로 나타난다.

label / required

label : label은 속성이 아니라 태그, 입력태그들에 대한 제목을 정의하는 태그

required : 항목에 내용이 비었을 경우 경고메세지 표시하는 속성





textarea

textarea 태그는 여러줄의 글을 입력할 수 있는 텍스트 상자다. textarea 태그는 인라인 요소이다.
