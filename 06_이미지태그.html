img 태그

img 태그는 이미지를 연결시켜 문서에 보이도록 해주는 태그

img 태그는 닫는 태그가 없다

링크연결은 href 속성, 이미지는 src 속성으로 경로를 작성한다.

alt 속성으로 이미지를 설명하는 글을 작성한다.

alt 속성에 작성된 내용은 엑박이 뜨거나 할 때 글자로 표시해주는 속성이다.

img 태그는 인라인요소. 가진것 만큼 너비가 들어오고, 줄바꿈이 일어나지 않는다.



작성중인 html 파일과 동일한 경로에 폴더가 있고, 그 폴더 안에 이미지를 불러올 경우

src="./폴더이름/파일이름.확장자"


작성중인 html 파일과 동일한 위치에 이미지를 불러올 경우

src="./파일이름.확장자"


작성중인 html 파일과 동일한 경로에 폴더가 있고, 그 폴더 안에있는 다른 폴더에 이미지를 불러올 경우

src="./폴더이름/폴더이름/파일이름.확장자"


작성중인 html 파일보다 상위 폴더 안에있는 이미지를 불러올 경우

src="../파일이름.확장자"


img 태그에 가로와 세로값을 줄 때 : width="수치" / height="수치" 라고 속성을 넣어서 사이즈를 조절할 수 있다.

이 때 수치의 기본 단위는 픽셀(px)이다.

수치를 줄일 때 너비든 높이든 한쪽만 줄이면, 그 수치에 맞게 정비율로 줄어든다.

강제로 크기를 조절해야 할 경우도 있지만, 통상은 너비, 높이 하나만 건드려서 자연스럽게 만들어 주는게 좋다.
