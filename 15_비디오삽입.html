video / source

  1. 4k 비디오 다운로드 프로그램을 이용
  2. 유튜브 등 비디오가 있응 사이트의 링크를 복사(화면 최상단의 링크 주소 등등)
  3. 4k 비디오에서 링크복사 버튼 클릭 > 다운로드 (용량과 화잘을 고려해서.. 저화질은 용량이 작지만 화질이 나쁘고, 고화질은 화질은 좋아지지만 용량이 커진다.)
  4. video 태그를 열고 닫기
  5. video 태그 안에 source 태그 열기 > src="으로 경로 지정
  6. vide 태그의 속성
    • controls : 컨트롤 패널을 표시해주는 속성(재생, 일시정지 등등 같은 것들 ..)
    • width / height : 영상의 가로 / 세로 사이즈를 정해주는 속성
    • autoplay : 자동재생 시켜주는 속성
    • muted : 음소거 시키는 속성
    • Ex : 켜자말자 자동재생되는 영상만 나오는경우(정지도 안되고 그냥 영상만 계속 재생되는 경우) : autoplay와 muted를 함께 사용한다.
    • loop : 반복재생 시켜주는 속성
